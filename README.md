\# Content-Based Image Retrieval



Project 2: Computer Vision Course



\## Description

Image retrieval system using classic features and deep network embeddings.



\## Features

\- Baseline matching (7x7 center square + SSD)

\- Histogram-based features

\- Texture features

\- ResNet18 embeddings



\## Build Instructions

```

mkdir build

cd build

cmake ..

cmake --build .

```
